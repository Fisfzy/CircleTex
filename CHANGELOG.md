# 更新记录

## 0.9.1

- 区域框选图片改为粗选吸附：从 PDF.js 页面绘制指令提取 `paintImageXObject` 的变换矩阵，计算嵌入图片在 PDF 页面中的真实矩形；
- 粗选框可只覆盖图片主体的一部分，也可包含图注空间；命中后自动吸附到完整图片盒子，图注不进入最终轮廓；
- 多张图片同时命中且评分接近时拒绝猜测，提示缩小选框；无图片绘制对象时保留明确错误，不使用像素轮廓误识别；
- 后端在确定唯一 `includegraphics` 后执行 SyncTeX 正向位置复核，PDF.js 吸附边界与源码命令位置不一致时拒绝调整；
- 定位期间显示淡灰色粗选虚线和蓝色吸附轮廓；定位成功后显示灰色原始边界与蓝色候选边界，上下箭头调整时只移动候选轮廓；
- 图片边界随 PDF 缩放、滚动和虚拟页面重渲染按页面坐标恢复，不依赖浏览器像素位置；
- 增加部分框选、包含图注、并排图片歧义、无命中、坐标容差、SyncTeX 边界复核和真实 PDF.js 绘制指令浏览器回归，单元测试增至 139 项。

## 0.9.0

- 复用现有区域框选支持图片尺寸调整；无文字或明显以图形为主的区域进入图片定位，不增加新的工具栏按钮；
- 使用中心、四角和边缘中点执行多点 SyncTeX 定位，并结合 `main.tex` 附近图片命令、真实图片文件与正向空间位置消歧；无法唯一确认时拒绝猜测；
- 在图片选框旁增加 `↑` 放大、`↓` 缩小、确认和取消控件，每次调整 5%，前端即时显示候选框，不触发 XeLaTeX；
- 确认图片调整只加入统一待编译队列；现有撤销、清空、查看改动和“应用 N 项并编译”同时管理文字与图片编辑；
- 首版支持单一 `width`、`height`、`scale`，无显式尺寸时增加估算的 `width=...\\linewidth`，简单子图中优先修改外层 `subfigure` 宽度；
- 图片参数写入复用源码基线哈希、范围重叠检查、备份、隔离编译、失败回滚和 PDF 刷新事务；图片命令变化或队列过期时拒绝应用；
- 复杂尺寸表达式、同时设置多重尺寸、缺失图片、自定义图片宏及无法确认的相邻图片继续保持不可直接调整；
- 增加图片负载、解析、普通图片、无尺寸、简单子图、复杂参数拒绝、联合队列和 Webview 浮动控件回归测试，单元测试增至 133 项。

## 0.8.4

- 文字／光标工具与区域框选工具改为可关闭的三态选择工具：再次单击当前工具进入未启用状态，处理方式、已有选区和待编译队列不受影响；
- 未启用工具时保留浏览器原生 PDF 文字选择用于复制，但不建立 CircleTeX 映射；非空直接编辑草稿继续阻止工具切换，避免输入意外丢失；
- 普通文字选区增加有序分页片段，每段携带独立页码、文字和 PDF 起止坐标，SyncTeX 按真实页面分别定位，不再把跨页终点错误发送到起始页；
- 跨页选区支持 Agent 分析及直接编辑的整体替换、删除；源码侧仍生成一项连续、原子的修改，拒绝源码顺序倒退、其他源文件、超过 120 行或文本无法安全匹配的范围；
- 待编译编辑矩形增加可选页码，多页草稿与红色预览分别绘制到对应页面，插入标签只显示在实际起止锚点页；
- 跨页拖动期间固定涉及页面，防止连续 PDF 虚拟渲染回收起始页文字层；最多支持连续 12 个页面，并过滤页眉、页脚边缘文字；
- 增加三态关闭、关闭后复制、分页负载校验、跨页 SyncTeX 页码路由、跨页直接替换和多页覆盖层回归，单元测试增至 126 项。

## 0.8.3

- 将“直接编辑 / Agent”处理方式与“文字光标 / 区域框选”选择工具拆为独立状态，工具栏新增上下排列的处理方式切换控件；切换任一维度不再隐式改变另一维度；
- 直接编辑支持区域框选，可在悬浮输入框中替换整个区域，或通过 `Backspace` / `Delete` 删除整个区域；区域不提供含义不唯一的前插和后插；
- 区域负载增加按视觉行和文字簇排序的片段、PDF 起止点及矩形，后端执行数量、顺序、边界、总长度和选区文字一致性校验；
- 增加区域源码解析器，只接受唯一且连续的普通正文范围，允许安全格式命令参数正文，拒绝跨公式、引用、未知命令、注释、表格、逐字环境和离散源码片段；
- 重复连续候选复用 SyncTeX 正向空间消歧；无法明显区分时继续拒绝，不接受前端提供源码偏移；
- 模式或工具切换不会清空待编译队列；非空直接编辑草稿会阻止切换，迟到的旧交互映射由状态版本丢弃；
- 增加四种处理／选取组合、区域替换删除、非空草稿保护、窄窗口切换控件以及区域安全范围回归，单元测试增至 123 项。

## 0.8.2

- 增加 LaTeX 可见结构投影，将普通正文、透明格式命令、公式、引用、未知命令、注释、数学环境、表格和逐字内容分开处理；
- 光标插入不再要求 PDF 两侧文字与整段源码连续相同；允许跨过已识别的行内公式和引用，稳定定位其前后的普通正文边界；
- `\\textbf`、`\\emph`、`\\underline` 等安全格式命令参数中的普通正文支持插入、删除和拖选替换，同时保留命令本身；
- 光标选区新增实际 PDF 坐标；多个安全源码候选复用 SyncTeX 正向空间消歧，只在最佳候选明显唯一时自动采用；
- 光标方向删除可在结构附近删除普通文字，但不会跨过公式、引用或未知命令，也不会把一个 PDF 数学字符映射为部分 LaTeX 源码；
- 公式内部、未知宏参数、注释、数学环境、表格和逐字内容继续保持不可直接编辑；失败时保留输入草稿，并返回更具体的结构安全提示；
- 增加公式前后、引用后、格式命令正文、重复候选、公式内部、未知命令和光标坐标回归测试，单元测试增至 117 项。

## 0.8.1

- PDF 审阅窗口启动时优先创建当前页壳体，不再等待全部页面尺寸扫描；其余页面元数据在后台补齐；
- 缓存最近稳定页的 JPEG 预览，并按 PDF 路径、修改时间和文件大小隔离版本；再次打开审阅时先显示缓存页，再由高清 Canvas 无缝接管；
- 缓存未命中时先串行生成低清当前页，随后生成高清 Canvas；高清页面显示后再在后台构建文字选择层，避免重图页长时间停留在加载界面；
- 外部 Skill 注册表初始化不再阻塞 PDF 审阅窗口创建，初始化完成后自动补发 Skill 列表；
- CircleTeX 日志增加 Webview、Worker、首屏壳体、低清预览、Canvas、文字层和后台元数据扫描耗时；
- 增加缓存预览先于页面壳体显示、高清接管后移除，以及 JPEG 缓存大小与格式回归测试。

## 0.8.0

- 增加 CircleTeX 自主管理的外部 Skill 注册表：导入包含 `SKILL.md` 的目录，执行静态检查、内容哈希快照、启用停用、更新移除和最近 50 条任务历史；
- 导入与更新向导要求确认任务类型、作用范围、输入快照、允许输出扩展名、声明命令、网络禁用、Codex 支持范围和超时；Skill 内容或权限变化后重新授权；

- 产物发布到 `exports/circletex/<skill>/<时间戳>/` 并保存 `run.json`，增加导入、权限、隔离、取消、产物校验和 Webview 选择流程回归测试。

## 0.7.0

- 增加 PDF 铅笔工具，支持拖选后键盘替换、`Backspace`/`Delete` 删除、单击文字间隙插入，以及光标前删和后删；

- 增加可选“查看改动”，直接模式不再强制打开差异页或二次确认，仍保留备份、基线校验、后台编译和失败回滚；
- 保留可选的修订痕迹模式；检测到 `0.6.0` 遗留修订时，要求先接受全部或拒绝全部，禁止与直接编辑混合写入；
- 光标定位改用唯一文字上下文和 Unicode 字素簇边界，完整处理组合重音、Hangul Jamo、变体选择符和 ZWJ 序列；
- 扩展公式、表格和逐字环境保护，拒绝在 `equation`、`align`、`tabular`、`verbatim` 等不安全上下文中直接编辑；
- 左侧设置增加 PDF 编辑模式选择，并对待编译队列锁定写入模式，配置切换不会改变已有队列语义；
- 编译入口拒绝未保存的 `main.tex`，产物覆盖后、事务提交前会再次校验磁盘源码和编辑器状态；校验失败时恢复旧 PDF 与 SyncTeX。
